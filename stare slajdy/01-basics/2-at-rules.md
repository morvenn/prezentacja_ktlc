# Regułki z małpką
![Avatar NFT Bored Ape należący do Madonny](gifs/ape.png)


## Rodzaje regułek

1. Zwykłe
   ```css
   @charset "utf-8"
   ```
2. Zagnieżdżone
   ```css
   @media print {
    /* regułki... */
   }
   ```
3. Warunkowe (sprawdzają własności przeglądarki)
   ```css
   @supports (someProperty) {
    /* regułki... */
   }
   ```
